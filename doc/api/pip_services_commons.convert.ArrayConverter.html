class pip_services_commons.convert.ArrayConverter.ArrayConverter¶

Bases: object

Converts arbitrary values into array objects. Example:

[code] value1 = ArrayConverter.to_array([1, 2]) // Result: [1, 2] value2 = ArrayConverter.to_array(1) // Result: [1] value2 = ArrayConverter.list_to_array(“1,2,3”) // Result: [“1”, “2”, “3”] [/code]
static list_to_array(value)¶

Converts value into array object with empty array as default. Strings with comma-delimited values are split into array of strings.

Parameters:value – the list to convert.
Returns:array object or empty array when value is None
static to_array(value)¶

Converts value into array object with empty array as default. Single values are converted into arrays with single element.

Parameters:value – the value to convert.
Returns:array object or empty array when value is None.
static to_array_with_default(value, default_value)¶

Converts value into array object with specified default. Single values are converted into arrays with single element.

Parameters:
  • value – the value to convert.
  • default_value – default array object.
Returns:

array object or default array when value is None.
static to_nullable_array(value)¶

Converts value into array object. Single values are converted into arrays with a single element.

Parameters:value – the value to convert.
Returns:array object or None when value is None.
