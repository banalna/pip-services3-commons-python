class pip_services3_commons.data.ProjectionParams.ProjectionParams(values=None)¶

Bases: list

Defines projection parameters with list if fields to include into query results.

The parameters support two formats: dot format and nested format.

The dot format is the standard way to define included fields and subfields using dot object notation: <code>”field1,field2.field21,field2.field22.field221”</code>.

As alternative the nested format offers a more compact representation: <code>”field1,field2(field21,field22(field221))”</code>.

Example:

filter = FilterParams.fromTuples(“type”, “Type1”) paging = PagingParams(0, 100) projection = ProjectionParams.from_value(“field1,field2(field21,field22)”)

myDataClient.get_data_by_filter(filter, paging, projection)
