class pip_services_commons.refer.Descriptor(group, type, kind, name, version)¶

Bases: object

Component descriptor used to find a component by its descriptive elements: - logical group: package or other logical group of components like ‘pip-services-storage-blocks’ - component type: identifies component interface like ‘controller’, ‘services’ or ‘cache’ - component kind: identifies component implementation like ‘memory’, ‘file’ or ‘mongodb’, … - component name: identifies component internal content, … - implementation version: ‘1.0’, ‘1.5’ or ‘10.4’

Matches this descriptor to another descriptor exactly.

descriptor: another descriptor to match this one.

Returns: True if descriptors match or False otherwise.
