class pip_services_commons.convert.FloatConverter.FloatConverter¶

Bases: object

Converts arbitrary values into float using extended conversion rules: - Strings are converted to float values - DateTime: total number of milliseconds since unix epoсh - Boolean: 1 for true and 0 for false

Example:
value1 = FloatConverter.to_nullable_float(“ABC”) // Result: None value2 = FloatConverter.to_nullable_float(“123.456”) // Result: 123.456 value3 = FloatConverter.to_nullable_float(true) // Result: 1 value4 = FloatConverter.to_nullable_float(datetime.datetime.now()) // Result: current milliseconds
