class pip_services_commons.commands.Event.Event(name)¶

Bases: pip_services_commons.commands.IEvent.IEvent

Concrete implementation of IEvent interface. It allows to send asynchronous notifications to multiple subscribed listeners.

Example:

event = Event(“my_event”)

event.addListener(myListener)

event.notify(“123”, Parameters.fromTuples(“param1”, “ABC”, “param2”, 123)

See IEvent, IEventListener
